# NutriHouse

**NutriHouse** è un gestionale backend pensato per supportare l’attività di nutrizionisti, con funzionalità scalabili e predisposte per l’integrazione con frontend sia in modalità monolitica che tramite architettura a microservizi.

## 📌 Tecnologie utilizzate

- Java 21
- Spring Boot
- Spring Security + JWT
- JPA/Hibernate
- MySQL
- Maven

## ⚙️ Funzionalità principali

- Gestione utenti e autenticazione con JWT
- Gestione pazienti
- Gestione alimenti
- Creazione e gestione di diete giornaliere e piani alimentari
- Associazione alimenti a pazienti
- Configurazione flessibile dei ruoli (Admin, Nutrizionista)
